belajar heading di html

heading di html ada enam,yakni h1,h2,h3,h4,h5,h6 heading berfungsi untuk membuat judul untuk artikel dan juga sub judul pada bagian artikel.

membuat sub judul

sub judul atau sub heading dimulai dengan tag h2.lalu di ikuti dengan tag selanjutnya hingga sampai h6. sementara itu,tag hanya digunakan untuk judul artikel saja.
